ChocoTabe - But...

Artist : ChocoTabe
Title : But...
Date : 08.06.2019

Description

ChocoTabe의 29번째 자작곡. 'But...'는 상대에 대한 생각이 마음속에서 떠나지 않는 것을 표현한 음악이다. Electornic 장르에 BPM 118로, 적당한 템포이다. The Idolm@ster Million Live!의 바바 코노미의 솔로곡인 To...라는 노래에서 영감을 받은 음악이다. 그렇기 때문에 To...와 음악의 구조와 분위기가 비슷하다는 것이 특징이다.
